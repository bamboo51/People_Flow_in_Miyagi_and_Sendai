# 宮城県と仙台市における人流データの分析

## 概要

この分析は宮城県と仙台市における人流データに分析である。両方に分け、宮城県における人流データの可視化と仙台市における人流データの分析である。さらに、仙台市の人流データ分析は、日付（曜日）・天気・場所によって分け、分析した。それぞれの科目に相当するフォルダを選択し、分析結果をご覧できる。
